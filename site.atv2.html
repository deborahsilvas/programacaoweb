React Native

Aprenda uma vez, escreva em qualquer lugar.

Crie aplicativos nativos para Android, iOS e muito mais usando React.

React Native traz os melhores recursos do desenvolvimento com React para o desenvolvimento nativo.

É uma biblioteca JavaScript de primeira linha para a criação de interfaces de usuário.

Escrito em JavaScript, renderizado com código nativo. Os elementos primitivos do React são renderizados na interface do usuário da plataforma nativa, o que significa que seu aplicativo usa as mesmas APIs da plataforma nativa que outros aplicativos.

Desenvolvimento nativo para todos

React Native permite criar aplicativos verdadeiramente nativos sem comprometer a experiência do usuário. Ele fornece um conjunto essencial de componentes nativos independentes de plataforma, como View, Text e Image, que se relacionam diretamente com os blocos de construção da interface do usuário nativa da plataforma.

Comece com o pé direito usando uma estrutura.

React Native traz o paradigma de programação React para plataformas como Android e iOS. Para criar um novo aplicativo com React Native, recomendamos um framework como o Expo.

Roteamento baseado em arquivos

Crie telas em pilha, modais, de gaveta e com abas com o mínimo de código repetitivo usando seu sistema de arquivos.

Utilize qualquer biblioteca, SDK ou código nativo.

Gere alterações nativas ou escreva seu próprio código nativo. Use mais de 50 módulos para criar seu aplicativo.

Ferramentas de desenvolvedor

Comece rapidamente com o Expo Go e, em seguida, continue com o expo-dev-client para funcionalidades nativas.

Assista e aprenda

Assista às palestras da equipe React e aprenda a tirar o máximo proveito do React Native. Encontre as últimas novidades no Bluesky e no X.
