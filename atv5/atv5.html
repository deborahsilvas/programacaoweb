Resumo – Flexbox e Grid CSS

Flexbox

Sistema de layout unidimensional (linha ou coluna). Ideal para organizar e alinhar itens em sequência.

Principais funcionalidades

  • display: flex – ativa o Flexbox no container.
  • flex-direction – define a direção (row/column).
  • flex-wrap – permite quebrar a linha.
  • justify-content – alinha itens no eixo principal.
  • align-items – alinha no eixo cruzado.
  • align-content – alinha grupos de linhas.
  • flex-grow – define quanto cada item cresce.
  • flex-shrink – define quanto encolhe.
  • flex-basis – tamanho inicial do item.
  • flex – shorthand (grow, shrink, basis).
  • order – muda a ordem dos itens.
  • align-self – alinhamento individual.

Vantagens

  • Simples para layout em linha/coluna.
  • Alinhamento fácil (inclusive centralização).
  • Excelente para componentes (menus, cards, listas).
  • Itens crescem/encolhem automaticamente.
  • Alteração de ordem sem mexer no HTML.

CSS Grid

Sistema de layout bidimensional (linhas e colunas). Ideal para estruturar páginas completas.

Principais funcionalidades

  • display: grid – ativa o Grid.
  • grid-template-columns / rows – define colunas e linhas.
  • grid-template-areas – cria áreas nomeadas.
  • gap – espaçamento entre células.
  • grid-auto-flow / auto-rows / auto-columns – controla células automáticas.
  • grid-column / grid-row – posiciona o item.
  • justify-content / align-content – alinha o grid dentro do container.
  • justify-items / align-items – alinha itens dentro das células.
  • justify-self / align-self – alinhamento individual.

Vantagens

  • Controle total sobre linhas e colunas.
  • Cria layouts complexos e responsivos facilmente.
  • Áreas nomeadas tornam o layout mais legível.
  • Perfeito para estrutura geral da página.
  • Permite sobreposição de elementos.
